# Assets Folder

This folder contains the company logo and other static assets.

## Logo File

Please place the YNM Safety logo file here with the filename: `ynm-logo.png`

The logo should be:
- PNG format (or SVG for better scalability)
- High resolution (at least 400x400px for best quality)
- Transparent background (if possible)

The logo will be automatically displayed on:
- Login/Signup page (centered above the form)
- All page headers (top-left, next to page title)

If the logo file is not found, the pages will still function normally without displaying the logo.
